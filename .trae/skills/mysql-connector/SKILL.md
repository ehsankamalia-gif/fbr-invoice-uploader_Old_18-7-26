# MySQL Connector

A skill for connecting to and querying the local MySQL database for the FBR Invoice Uploader project.

## Features

- List all tables in the database
- Execute custom SQL queries
- Inspect table structure
- View data with optional limits

## Usage

Simply invoke this skill when you need database access!
